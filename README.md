Longley Dataset Regression AnalysisThis repository contains an end-to-end statistical analysis of the classic Longley macroeconomic dataset using the R programming language. The project demonstrates data exploration, correlation analysis, simple linear regression modeling, model selection, and verification of parameters using raw matrix algebra.Technical OverviewThe analysis follows a structured 5-step workflow designed to evaluate predictors for the total number of individuals Employed:Exploratory Data Analysis & Correlation:Computed the complete correlation matrix for the dataset to isolate variables impacting employment numbers.Visualized the relationships by plotting Employed against the top 3 most highly correlated macroeconomic variables (typically GNP, Year, and GNP.deflator).Regression Modeling & Champion Selection:Built separate simple linear regression models ($Y = \beta_0 + \beta_1X + \epsilon$) for each of the top 3 selected variables.Evaluated model fitness using Adjusted $R^2$ and Residual Standard Error (RSE) to select the best performing predictor ("Champion Model").Matrix Decomposition & Mathematical Verification:Formed the exact response vector $Y$ and the design matrix $X$ (incorporating a column of $1$s to account for the intercept).Recalculated the Ordinary Least Squares (OLS) estimator parameters ($\hat{\beta}$) using standard matrix equations:$$\hat{\beta} = (X^T X)^{-1} X^T Y$$Programmatically verified that the matrix-derived coefficients and fitted predictions perfectly matched the outputs generated by R's built-in lm() function.File Structurelongley_analysis.R: The core script containing data ingestion, automated variable selection, plotting loops, and the matrix calculations.RequirementsTo run this script locally, you only need a standard installation of R or RStudio. The dataset is built directly into R's native datasets package, meaning no external data downloads or heavy package dependencies are required.
